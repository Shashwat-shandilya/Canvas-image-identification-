Canvas image identifier

*You can draw the following objects - house, battle-field, tree, swimming pool.*


NOTE::You can also drag and drop items to draw and please refresh after identifying the drawing



NOTE::the below keys are commands to draw

  • shift + I - increasing height and width of blocks
  • shift + D - decreasing height and width of blocks
  • water - d
  • ground - g
  • cloud - c
  • light-green - l
  • stair1 - r
  • stair2 - shift + r
  • light-green - l
  • wall - w
  • yellow-wall - y
  • fireball - f
  • fireball2 - shift + f
  • angry - a
